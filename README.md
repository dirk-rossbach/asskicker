# Asskicker Features
* file based DB
	* player api
		* add player to team roster
			* name
			* profile pic
			* uuid
		* generate QR code for each new player
	*  match api
		* add player to match
		* auto round robin
		* add player to color
		* score goal per color
		* remove last (false) goal
		* start match
		* end match
	* web socket sync with all Fes 
* statistics
	* match started
	* match ended
	* goal scored
		* time
		* player
		* color
